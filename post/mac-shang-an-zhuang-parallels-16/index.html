Mac 上安装 Parallels 16

2021-05-15

最近收入一台macbook pro，简单记录一下过程，过程碰到了window10不能上网的问题，希望能帮助到后面的朋友。

一、下载安装好 parallels 16

二、安装免激活的 window10

三、修改配置文件

打开 Terminal，按照图片输入

cd  /Library/Preferences/Parallels
open .

会看到打开一下 Finder ，可以看到两个配置文件，网络相关的是 network 开头的

可以用mac 自带的文本编辑器来操作，但由于权限会提示先复制出来，后续调整后再覆盖进原有位置。

在这里，笔者建议使用 vs code 编辑器进行编辑，当保存时会提示不能保存，并提示输入密码进行保存

为解决网络不能正常连接的问题，需要替换"network.desktop.xml"文件中的"<UseKextless>1</UseKextless>"或者"<UseKextless>-1</UseKextless>"，把这一行修改为"<UseKextless>0</UseKextless>"并保存。

如果不存在这个配置项，那么可以 “<IPv6Enabled>1</IPv6Enabled>” 后面增加 “<UseKextless>0</UseKextless>”，即可。笔者，就是属于这一情况。

过程中，可能会要求安装 parallels 的系统插件，通过就可以。具体插件名称没有记录下来。

然后，重新启动机器。
