让我们开始吧

什么是流程控制

Rust 允许我们通过特定的表达式来执行条件，进而控制我们程序的流程。基于这些条件的结果，Rust 将会执行指定的代码块。

举个例子，让我们来考虑这么一个应用：特定领域中的成员，在一个人员获取到进行这领域的权限时，他们需要提供下正确的认证信息。

如果他们提供了正确的认证信息，他们就会被允许进入这区域。否则，他们将可以进行选择：再次尝试，或者当他们忘记密码时重置他们的密码。

基于这些逻辑，我们可以想象出我们的代码，类似下方：

if username == db_username && password == db_password
  redirect to member area
otherwise
  if forgot_password
    reset password
  try_again

我们真正的代码并不会像和上面的完全一样，但它可以作为给你们展示条件控制的示例。

控制我们程序流程的有下面的语句，可以结合比较和逻辑运算符进行操作。

  • if
  • else if
  • else
  • match

if 条件表达式

我们使用 if 语句来表示条件的执行，如果条件证明为正确的，编译器会执行 if 语句中的代码块。

如果条件证明是错误的，编译器会移动到 if 语句后的代码块。

if 语句通过 if 关键字来构建，后面接着的是我们想要执行的条件。

然后，我们在大括号的里面指定代码块。在这代码块里，我们可以编写一个或者更多的表达式，如果条件被证明是正确的，代码块中的代码将会被执行。

if condition {
    // code block to execute
    // if condition proves true
}

注意，如果一个语句没有用分号;进行结束，那么编译器会通过大括号的关闭符号来判断。

fn main() {
    let a = 5;
    if a < 10 {
        println("Evaluation proved true");
    }
}

在上面的例子中，我们的变量是小于10的，所以条件会证明是正确的。编译器会打印在 if 语句代码块的信息。

当编译器执行完成 if 语句(代码块)时，它会移动到后面的任意代码。这与 if 语句为错误时类似，编译器会移动下 if 语句外部下面的代码。

fn main() {
    let a = 5;
    if a > 10 {
        println!("Execute only if the condition proves true");
    }
    println!("Code outside and below if statement");
}

在上面的例子中，我们的 if 条件判断是错误的，编译器会移动到 if 语句下面的下一部分代码块中。

怎么处理多条件判断

通过逻辑运算符 && 和 || , 可以帮助我们在一个 if 条件中执行不止一个条件。“And” 可以用来判断一个条件和另外一个条件都是true，在执行代码之前，需要两个条件都证明为true。

为了使用 And 来处理多个条件，我们使用 && 操作符来分开两个条件。

if condition_1 && condition_2 {
    // code block to execute if both conditions prove true
}

fn main() {
    let a = 5;
    if a > 0 && a < 10 {
        println!("Both conditions are true);
    }
}

在上面的例子中，由于我们两个条件都是正确的，所以 if 代码块中的代码会被执行。

"OR" 条件用来判断一个条件或者另外一个条件证明 true, 在执行代码之前，需要多个条件中至少有一个证明为true。

为了处理在多条件中的 OR，我们使用 || 操作符来分隔两个条件。

if condition_1 || condition_2 {
    // code block to execute if one condition or the other proves true
}

fn main() {
    let a = 5;
    if a > 5 || a < 10 {
        println!("One of the conditions are true");
    }
}

在上面的例子中，第一个条件是false, 但第二条件是 true，所以编译器允许执行代码块中的代码。

else if 条件判断

当我们需要多种判断条件但又不能整合到一个条件表达式中时，可以使用 else if 来组合更多的 if 条件。

对于每一个 if 语句，当前一个 if 条件判断完成时，编译器会移动到下一个逐步处理的。

编写 else if，也就是在 if 语句之前，增加 else 关键字。

if condition_1 {
    //
} else if condition_2 {
    //
}

fn main() {
    if false {
        println!("Primary if statement");
    } else if true {
        println!("Secondary if statement");
    }
}

在上面的例子中，我们在第一个 if 表达式之后添加了第二个 if 表达式，中间使用 else 分隔。

_注意：else if 不能独立于 if 语句而存在，第一个条件判断必须是 if _

我们可以编写更多我们需要的 else if 语句。

fn main () {
    if false {
        println!("Primary if statement");
    } else if false {
        println!("Secondary if statement");
    } else if true {
        println!("Teritiary if statement");
    }
}

else 表达式

else 表达式可以处理所有 if 和 else if 表达式未能覆盖到条件。

因为它是“处理所有”的，所以它没有 条件判断语句，只有执行代码块。

if condition_1 {
    // 
} else {
    //
}

fn main() {
    let number = 5;
    if number == 10 {
        println!("if statement");
    } else {
        println!("catch all else callback");
    }
}

上面例子中，else 表达式会处理除了 number == 10 之外的结果，因为 number 不等于 10 ，所以编译器会执行 else 的代码块。
